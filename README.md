# GO Rest Demo

### Dependencies

```
$ dep status
$ dep ensure
```

### Run
```
$ docker-compose up 
$ go build && ./go-basics

navigate to: http://localhost:8000/users
navigate to: http://localhost:8000/groups
navigate to: http://localhost:8000/transactions
```
